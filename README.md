# ☕ Cafe Management System (Full-Stack PHP & MySQL)

A robust and user-friendly web application designed to streamline cafe operations, from menu management to order billing and inventory tracking.

## 🌟 Key Features
* **Admin Dashboard:** Manage menu items, prices, and view sales reports.
* **Customer Interface:** Simple and intuitive menu browsing and ordering.
* **Real-time Billing:** Automated invoice generation for customers.
* **Database Integration:** Secure storage of records using MySQL.

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP (Server-side scripting)
* **Database:** MySQL
* **Server:** XAMPP / Apache

## 📂 Database Design
This project uses a relational database structure to ensure data integrity.
* **Tables:** `Users`, `Menu_Items`, `Orders`, `Order_Details`.
* **Primary/Foreign Keys:** Implemented for efficient data linking.

## 🚀 How to Run
1. Clone the repository.
2. Import the `cafe_db.sql` file into your local phpMyAdmin.
3. Place the project folder in your XAMPP `htdocs` directory.
4. Open `localhost/cafe-management-system` in your browser.
